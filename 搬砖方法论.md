```csharp
生成每一个UI的打开方法，
如果OpenParams有多个，每个List<string>都是参数类名序列，分别生成每个方法，如果为空，只生成默认的方法;
方法名为Open_UIName()
默认方法为
public static int OpenUIForm_UIName()
{
   return GameEntry.UI.OpenUIForm(AssetUtility.GetUIFormAsset(UIName), GroupName);
}
类似 
public static int OpenUIForm_UIName( 参数..)
{
  return  GameEntry.UI.OpenUIForm(AssetUtility.GetUIFormAsset(UIName), GroupName,参数..);
}
```
